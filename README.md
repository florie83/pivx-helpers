Download and use:

```
cd ~/.pivx-cli
wget https://github.com/givanse/pivx-helpers/archive/master.zip
unzip master.zip
mv pivx-helpers-master/bin .
echo 'export PATH=$HOME/.pivx/bin:$PATH' >> ~/.bash_paths
source ~/.bash_profile
```
